## What is Trace OCR?

Trace OCR is a REST API for extracting text from images and PDFs. It combines deep learning-based text detection with text recognition to deliver structured OCR results with confidence scores and bounding box geometry.

## Quick example

```bash
curl -X POST https://ocr-api.trace.so/ocr/ \
  -F "files=@document.jpg"
```
